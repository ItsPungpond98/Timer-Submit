How to use it:
**Reserved Objectives:
timer, hscalc, big_multiply, big_add

**Reserved Storage:
timer:timesub, timer:timervalue

Timer Part
1. timer objective is reserved
2. Run the time:time/timer.mcfunction on repeat
3. Use the data from timer:timervalue to do stuff.

***You can't modify the world border when using the timer!

Time Submit Part
1. hscalc, big_multiply, big_add, objective is reserved
2. Run timesub/gen.mcfunction with the object {"v": #version, "map_id": #map_id}
**Version 1 is the 13 char code
**Version 2 is the 19 char code

3. You have your link!
